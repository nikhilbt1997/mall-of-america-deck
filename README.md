## Overview

A fully interactive, browser-based sales deck for Mall of America — built as a cinematic, luxury-grade pitch tool for prospective retail tenants, corporate sponsors, and event partners. Designed to work both as a live screen-share tool and a standalone link sent to prospects.

---

## Live Features

### Core Sections (Phase 1)
| Section | What It Does |
|---|---|
| **Hero** | Cinematic YouTube background video, animated headline, 3 primary CTAs |
| **Why MOA** | Animated CountUp stats, visitor demographics, MOA vs Standalone comparison table, Interactive Floor Map, Demographics Charts |
| **Retail** | 6 brand category breakdown, animated brand marquee, growth metrics |
| **Luxury** | Animated brand name marquee, premium positioning stats, editorial quote |

### Phase 2 Sub-Modules (All Built)
- **ROI Calculator** — select brand category + space size → estimated annual revenue vs standalone
- **Interactive Floor Map** — SVG property map with clickable zones navigating to sections
- **Event Booking Calendar** — 6-month visual calendar with clickable event dates

- **AI Prospect Qualifier** — brand details + goals → personalised MOA pitch generated by Claude
- **Smart Contact Form** — submit inquiry → Claude drafts personalised MOA team follow-up email

## Design System

**Color palette:**
- Near-black: `#080808` — primary background
- Warm gold: `#C9A84C` — accent, CTAs, highlights
- Gold light: `#E8C96A` — hover states, shimmer
- Pure white — text, headings

**Typography:**
- `Playfair Display` — editorial headings (luxury brand feel)
- `DM Sans` — refined body text
- `DM Mono` — data labels, eyebrows, stats

**Motion principles:**
- All entrances: `opacity 0 → 1` + `translateY(40px → 0)`
- Stagger delay: `i * 0.08s` across card grids
- Easing: `[0.16, 1, 0.3, 1]` (custom spring) for headlines
